Formules Mathématiques pour le Plan 2D

  • Distance entre deux points :
    d = sqrt((x2 - x1)^2 + (y2 - y1)^2)
  • Milieu d'un segment de ligne :
    M = ((x1 + x2) / 2, (y1 + y2) / 2)
  • Pente d'une ligne :
    m = (y2 - y1) / (x2 - x1)
  • Équation d'une ligne :
    y = mx + b
  • Surface d'un triangle :
    A = (1 / 2) * | x1(y2 - y3) + x2(y3 - y1) + x3(y1 - y2) |
  • Équation d'un cercle :
    (x - h)^2 + (y - k)^2 = r^2
